Objetivo:
Desarrollar una aplicación web que acceda a los datos suministrados por la API [1].

Requisitos:
No existe ningun requisito obligatorio o indispensable, aunque se valorará muy positivamente los siguientes puntos:
- Proyecto realizado con Angular5.
- Estilos basados en Material Design.
- Realización de Test unitarios y end to end.
- Guardado/modificacion de datos mediante la API [1]

Tiempo de Entrega:
No existe un tiempo de entrega o límite, conforme usted vaya teniendo su código disponible para ser evaluado, notifiquelo (véase siguiente punto). 

Forma de entrega:
Cuando usted tenga el código terminado, contacte con tomas@exo.works y adjunte el link de su cuenta de github para descargarlo. Sino dispone de cuenta, adjunte el proyecto en formato zip o tar.gz por email.

Dudas:
Si tiene dudas o algun problema, contacte con tomas@exo.works

[1]
El modelo de datos es muy sencillo, se trata simplemente de un listado de personas, con una serie de campos de distintos tipos y un conjunto de comentarios sobre esas personas. Todos los datos son ficticios.

Existen tres API:
http://demopeople.exolever.com/api/consultants/ --> Listado de la gente de la base de datos.
http://demopeople.exolever.com/api/consultants/<pk>/ --> Obtiene el detalle de una persona concreta.
http://demopeople.exolever.com/api/comment/ --> POST, crear un nuevo comentario sobre esa persona

Si accedes por una navegador web a esas URLs, podrás ver el JSON que se está generando, para obtener los campos de la API así como sus valores.
